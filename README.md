# Automatización de indicador

Este proyecto consiste en una automatización desarrollada para procesar información en Google Sheets, reducir trabajo manual y estandarizar la generación de resultados dentro del archivo de control.

## Objetivo

El objetivo principal de esta automatización es ejecutar de forma ordenada y automática una serie de pasos sobre un archivo de Google Sheets, incluyendo limpieza de datos, copiado de información, aplicación de fórmulas, clasificación y actualización de hojas de trabajo.

## ¿Qué hace esta automatización?

La solución realiza tareas como:

1. Limpieza de datos en rangos específicos sin afectar encabezados.
2. Copiado de información desde un archivo fuente.
3. Creación o actualización de estructuras necesarias dentro del archivo destino.
4. Procesamiento de columnas con fórmulas y lógica de negocio.
5. Clasificación automática de registros según reglas definidas.
6. Ejecución de rutinas de auditoría o validación final.

## Tecnologías utilizadas

- Google Sheets
- Google Apps Script
- GitHub para control de versiones

## Flujo general

1. Se limpia la información anterior en las columnas definidas.
2. Se trae la nueva información desde el archivo origen.
3. Se actualizan fórmulas o campos calculados.
4. Se clasifican los registros según la lógica del negocio.
5. Se ejecutan validaciones finales.
6. Se deja el archivo listo para uso operativo.

## Beneficios

- Reduce errores manuales.
- Ahorra tiempo operativo.
- Mantiene un proceso estandarizado.
- Facilita futuras mejoras y mantenimiento.
